# sandbox-react-form-validation-2022

## What's this?

Example implementation of hand-made react form validation.
This is just a proof-of-concept.

## Command

- install precommit hooks

  ```bash
  npm run prepare
  ```

- type checking

  ```bash
  npm run type-check
  ```

- lint

  ```bash
  npm run lint
  ```

- code formatting

  ```bash
  npm run fmt
  ```
